Eumpy.real_time_detection.EEG.EEG_feature_extract module¶

Created on Mon Oct 22 12:21:27 2018

@author: Yongrui Huang

This script is used for extracting average PSD (Power spectral density) features from raw EEG data. In the meantime, it also puts the features into hard disk in ‘.npy’ format.

The precess is described as follows. 14 channels and 3 asymmetric pair channels are picked. For each channel, we extract their PSD feature in each unit second(e.g. 1 second) with a 50% overlap.

Note: change tha name MAHNOB-HCI and you can extract your own data!

class Eumpy.real_time_detection.EEG.lstm_classification.LstmModelClf¶

Bases: object

This class is used to apply lstm model to classify the valence and arousal. The emotion state value (valence or arousal) ranges from 1 to 9. We consider it as two category. When it is higher than 5, we take it as high category. when it is less than or equal with 5, we take it as low category.

X_std: the standard deviation vertor for the feature

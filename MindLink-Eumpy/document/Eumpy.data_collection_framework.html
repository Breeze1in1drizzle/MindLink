Eumpy.data_collection_framework.start_GUI module¶

Created on Tue Oct 16 10:38:13 2018

@author: Yongrui Huang

This package provides a GUI for collecting online data in an emotion experiment.

The process of the emotion experiment is as follows.

The experiment consists of several trials.

  1. At the begining of each trial, there is a 10-second countdown in the

center of the screen to capture each subject’s attention and to serve as an indicator of the next clip.

  1. After the countdown is complete, movie clips that include different

emotional states are presented on the full screen. While the movie clips are palying, we record the signal of the subjects.

  1. After each trials ends, a SAM form will appear to evaluate the emotion

when the subject is watching clips.

Both of the singnal and emotion ground truth label will be saved in the

hard disk.

In this package, we collected EEG and facial expression data as example.

Here is some instruction of hwo to use this framework.

  1. If you just want to collect EEG and facial expression dataset. just run

start_GUI.py using commond ‘python start_GUI.py’ and it will start the experiment.

  1. If you want to use it in your own code. You can follow the example

presented in ‘start_GUI.py’. You first create the ‘DataCollection’ object and then add some ‘recorder’ to it. Finally, you call the method start(). As for the ‘recorder’, please see 3.

  1. If you want to record different singnal, for example, ECG, you will have

to create a new class who inheirs ‘AbstractRecorder’ in ‘util’. Please see the class document in ‘AbstractRecorder’, where it will give you more detail about how to create a recorder.

Here it will show you how this framework works.

  1. When you call start() method in the DataCollection class. It all begins.
  2. An personal information form comes out. It requires the test subject

for their personal information and those information will be saved as well.

  1. When the ‘start Testing’ button is clicked, one trial starts.
  2. In the beginning of each trials, a 10-s countdown page appears.
  3. After the countdown, the corresponding video is presented. In the mean

time, the recorder start recording the data.

  1. When the video is over, the recorder stops recording and a page with a SAM

form appears. This page requires the subject repoert their emotion status. After clicking ‘Submit’ button, if there is still some trial left, return to 4, else go to 7.

  1. A finish page will appear and 3-s countdown will start. When the countdown

is over, the GUI will close and the program will quit.

This script gives an example of how to use this data collection framework.
